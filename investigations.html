Instructional Design

Foundational Investigation

Each central activity will be called a foundational investigation. This will be available in two levels of challenge: Level 1 and Level 2.

Components

Each foundational investigation will have these components:

  • Two instructional videos that "bookend" the activity
  • Teacher and student versions of Jupyter notebooks
  • A teacher's guide
  • NGSS support and assessment materials
  • Class management tools

Extensions

Related investigations that build upon the successful completion of the foundational investigation will be called extensions.

Foundational Investigation Level 1

These address the "big ideas" covered in most introductory courses and align well with LSST Science Goals.

All code will be hidden from view in Level 1 Jupyter notebooks. Interactivity with data will be via widgets. Each activity will take between 1-2 hours to complete, depending on how much class discussion and assessment is included.

Foundational Investigation Level 2

This is an alternate pathway to achieving the same learning goals as Level 1, but the code will be partially revealed and its functionality explained. Students will be encouraged to read and manipulate variables within the code using a guided process. An optional challenge problem will be included at the end of the investigation, which will draw upon both science and coding aspects. The intent of Level 2 is to provide students with an introduction to understanding coding and its relevance to modern science practice, but not to teach coding. The instructional videos, teacher's guide, and miscellaneous support materials will be the same as Level 1.

Extension Investigations

These are complete notebook investigations that build upon prior learning in a foundational investigation. They will feature teacher and student Jupyter notebooks, a teacher's guide and support materials, but will not have instructional videos. They will be available only as Level 1 notebooks. The intent of extensions is to explore a topic deeper, but the content is not necessarily more challenging. Each extension will take between 1-2 hours to complete.
